Random Word

Random word is a new feature introduced with 0.4Beta. Random word feature allows user to have a window that can show random words from telugu and english.

Random word improves vocabulary by letting user to see through a random dictionary word (both telugu and english) at a time.

Random word options:

  • Always On Top: Shows the random word always on top of the screen. Accessible via right click on tray icon.
  • Timer setting: Gives user option to set the random word display timer. After configured time interval, a new word will be displayed from telugu and english.
  • Opacity setting:Random word display can be set with a defined opacity level from user
  • Playback controls:Random word display will be shown with standard controls "Forward, Play/Pause, Stop, and Rewind". These controls will let user to have the random word display pause for some time and playback to previous words.
